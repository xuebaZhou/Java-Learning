java知识点：

1、Java是一种严格的类型语言，它不允许数值类型和布尔类型之间进行转换。

2、char类型的常量值必须用一对单引括号(’ ’)括起来。
   例. char ch1=‘a’, ch2=‘1’, ch3=‘\t’

3、Float.NaN 表示“不是一个数字”（Not a Number），它是 Java 中 float 类型的特殊值之一。
NaN 在浮点运算中有一些特殊的性质：
任何数与 NaN 进行任何运算的结果都是 NaN。
NaN 不等于任何数，包括它自己。

4、浮点数常量是double型的，除非用字母f明确说明它是float型的。
	示例：
		5.31 //双精度浮点型常量
		5f //单精度浮点型常量
		0.0124e+6 //双精度浮点型常量
按值传递
➢ 当参数的数据类型是基本数据类型时，实参和形参
之间是按值传递。
➢ 按值传递是指调用方法将实参的“值”传递给形参，
也就是说被调用方法中形参的值是调用方法指定的
值的拷贝。
➢ 按值传递时，实参和形参在内存中占据不同的空间，
因此当实参的值传递给形参后，两者之间将互不干
扰，形参值的改变不会影响原来实参的值
